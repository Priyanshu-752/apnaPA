## File Naming

```text
YYYYMMDD_HHMM_description.md
```

```text
20260513_1114_context_pipeline_setup.md
```

## Files Modified

- `path` - change

## Files Deleted

- `path` - reason

## Decisions

- Decision and rationale.

## Issues

- Issue and resolution.

- Next action.
