# apnaPA Context Checkpoints

Timestamped session snapshots live here. Use the latest checkpoint to resume work without relying on chat history.

---

## File Naming

```text
context_checkpoint_DD-MM-YYYY-H-MMPM.md
```

Example:

```text
context_checkpoint_13-05-2026-11-14AM.md
```

---

## Checkpoint Template

```markdown
# Context Checkpoint - DD-MM-YYYY H:MMAM

## State

Current project status.

## Completed This Session

- Completed item.

## In Progress

- Active item.

## Next Steps

- Next action.

## Important Context

- Details the next session should know.
```
