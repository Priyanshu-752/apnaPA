# apnaPA Audits

Audit reports capture review prompts, scope, findings, and follow-up actions.

---

## Common Audit Scopes

- Security and auth.
- Prompt injection and AI write confirmation.
- Data isolation by user id.
- Database schema and migrations.
- Telegram linking and webhook guardrails.
- n8n workflow ownership boundaries.
- Dashboard usability and accessibility.
- Test coverage and reliability.

---

## File Naming

```text
audit_YYYYMMDD_HHMM_scope.md
```
